
브루트포스 - N과M
=======================  
브루트 포스에서 방법을 만드는 방법은 주로 3가지가 있다
1. 재귀 
2. 순열 
3. 비트마스크
  
그 중에서도 재귀함수가 제일 중요한데 왜냐하면 순열과 비트마스크는 재귀로 변환시킬 수 있기 때문이다.   
   
재귀를 사용하는 브루트 포스는 대개 2가지로 나눌 수 있는데
1. 순서와 관련된 문제  
2. 선택과 관련된 문제  
  
**문제 풀이에 앞서 N과M의 모든 문제는 사전순으로 증가하는 형태로 출력해야한다!!!!**     
# 1. N 과 M (1)
* 1부터 N까지 자연수 중에서 **중복 없이** M개를 고른 수열을 모두 구하는 문제  
* 1 <= M <= N <= 8
예를 들면 N이 5이고 M이 3이면?
```
123
124
125
132
134
135
.
.
.
```
이렇게 중복 없이 m개를 고르는 문제는 순서와 관련된 문제라고 볼 수 있다.    
왜 그렇냐면 각각의 순서마다 들어간 값을 알아야 다음 수를 알 수 있기 때문이다.     
만약 중복이 있었다면 앞 순서에 어떤 값이 오더라도 뒤 값은 영향을 받지 않기에 순서가 아니라고 볼 수 있다.     
       
여기서 우리가 할 수 있는 가장 단편적인 방법은 앞에서 부터 수를 하나씩 넣는 것이다.    
그리고 그다음 숫자는 앞에서 정한 숫자를 제외한 숫자들이 경우의 수일것이다.    
     
중복없이를 구현하기 위해서 우리는 사용된 숫자라는 것을 알리기 위한 배열 C를 만들어 사용할 것이다.  
```
bool c[10]
```
```c[i]``` 값이 true 이면 i라는 값을 사용한 것이고 반대로 false 이면 사용하지 않은 것을 의미시킨다.  
또한 ```a[]```을 이용해서 각 자리수마다 저장된 숫자들을 넣어줄 수 있다.  
```
int a[10]
```

```c++
#include <iostream>
#include <algorithm>
#include <vector>
#include <string>
#include <stack>
#include <queue>
#include <deque>

using namespace std;

bool c[10]; 
int a[10];
void go(int index, int n, int m){ // m은 자리수를 의미
	if(index == m) {
		for(int i=0; i < m ;i++){
			cout << a[i];
            if (i != m-1) cout << ' ';
		}
		cout << "\n";
		return; // 탈출 조건
	}
	for(int i=1; i <=n; i++){
		if(c[i]) continue; 
		c[i] = true;
		a[index] = i;
		go(index+1, n, m);
		c[i] = false; // 이전 것을 풀어줘야 다음것을 할 수 있음
					  // 한 순서에 1개만 선택할 때 사용할 수 있는 방법이다.  
	} 
}

int main(void){

	int n,m;
	cin >> n >> m;
	go(0, n,m);
	return 0;
}
```
    
# 2. N 과 M (2)
* 1 부터 N까지 자연수 중에서 중복 없이 M개를 고른 수열을 모두 구하는 문제(오름차순)   
* 1 <= M <= N <= 8
```
123
124
125
134 같이 전 숫자보다 큰 숫자가 와야 한다.  
```
  
**소스 코드1**
```c++
#include <iostream>
#include <algorithm>
#include <vector>
#include <string>
#include <stack>
#include <queue>
#include <deque>

using namespace std;

bool c[10]; 
int a[10];
void go(int index, int start, int n, int m){ // m은 자리수를 의미
	if(index == m) {
		for(int i=0; i < m ;i++){
			cout << a[i];
            if (i != m-1) cout << ' ';
		}
		cout << "\n";
		return; // 탈출 조건
	}
	for(int i=start; i <=n; i++){
		if(c[i]) continue; // 전보다 작으면 continue
		c[i] = true; 
		a[index] = i;
		go(index+1,i+1, n, m);
		c[i] = false; // 이전 것을 풀어줘야 다음것을 할 수 있음
					  // 한 순서에 1개만 선택할 때 사용할 수 있는 방법이다.  
	} 
}

int main(void){

	int n,m;
	cin >> n >> m;
	go(0,1, n,m);
	return 0;
}
```
다른 방법으로도 풀어 볼 수 있는데 바로 오름차순이란느 특징을 이용하는 것이다.  
예를 들면 1,3,5 를 가지고 숫자를 만들게 된다면 
```
135
153
315
351
513
531
```
이렇게 숫자를 만들 수 있지만 사실상 오름차순으로 나타내면 ```135```만 사용할 수 있다.      
그래서 이번에는 순서가 아닌 '선택'을 한다는 개념으로 접근해보자       
           
1부터 N까지의 숫자가 존재한다고 하면 각각은 포함될 수 도 포함되지 않을 수도 있다.    
```
 1 2 3 4 5 6 7 8 9
 o o o o o o o o o
 x x x x x x x x x 
```
이를 구현하려면 각각의 수를 포함하는 경우, 선택하는 경우 ,선택하지 않은 경우로 나누어 작성하면 된다.  
포함한다는 개념은 현 상태로는 이해가 어렵지만 코드를 보면 알 수 있다.  
우선 내 기준으로 기술해보자면 이전에 start 처럼 지나간 숫자는 사용하지 않는 것이 포함하는 경우를 의미한다.  
즉 1부터 한칸씩 증가되니까 2로 갈 경우 1은 포함되지 않는 것이다.    
이렇게 포함되는 수 또는 현재 위치의 수를 의미하는 index라는 변수를 만들어 사용할 것이고  
선택에도 갯수 제한이 있으므로 이에 대한 selected 변수를 만들 것이다.  
즉 ```변수[selected] = index``` 의 형태로 값이 저장될 것이다.  

**소스 코드2**
```c++
#include <iostream>
#include <algorithm>
#include <vector>
#include <string>
#include <stack>
#include <queue>
#include <deque>

using namespace std;

int a[10];
/// index 는 1부터 N까지 가는길에 현재 위치 
/// selected 는 선택된 수
void go(int index, int selected, int n, int m){ // m은 자리수를 의미
	if(selected == m) {
		for(int i=0; i < m ;i++){
			cout << a[i];
            if (i != m-1) cout << ' ';
		}
		cout << "\n";
		return; // 탈출 조건
	}
	if(index > n) return; // 마지막까지 갔는데 selected 가 안차면 그 이상으로 가니까 종료시키기
	//////// 선택했을때의 경우의 수 //////// 
	a[selected] = index;
	go(index+1, selected+1, n ,m);
	////////////////////////////////////////

	//////// 선택 안했을때의 경우의 수 //////// 
	a[selected] = 0; // 선택안했다는 가정을 하기 위해 위해서 index 넣은 것을 초기화 
	go(index+1, selected, n ,m);
	///////////////////////////////////////////
}

int main(void){
	int n,m;
	cin >> n >> m;
	go(1,0,n,m); // 시작이 1부터 하므로
	return 0;
}
```

# 3. N 과 M (3)
* 1부터 N까지의 자연수 중에서 M개를 고른 수열을 모두 구하는 문제 **(중복 선택 가능)**
* 1 <= M <= N <= 7   
      
단순히 말하면 1의 코드에서 중복만 제거해주면 된다.     
**소스코드**   
```c++
#include <iostream>
#include <algorithm>
#include <vector>
#include <string>
#include <stack>
#include <queue>
#include <deque>

using namespace std;

int a[10];
void go(int index, int n, int m){ // m은 자리수를 의미
	if(index == m) {
		for(int i=0; i < m ;i++){
			cout << a[i];
			if (i != m-1) cout << ' ';
```

# 4. N 과 M (4)   
* 1부터 N까지 자연수 중에서 M개를 고른 수열을 모두 구하는 문제 (중복 선택 가능, 비내림차순)   
* 1 <= M <= N <=8   

비내림차순은 내림차순이 아닌것을 의미하며 중복된 숫자가 나와도 되는 오름차순이라 보면 된다.   
즉, 2번 코드에서 중복을 허용해주기만 하면 된다.  

**소스코드1**
```c++

#include <stack>

	if(index == m) {
		for(int i=0; i < m ;i++){
			cout << a[i];
            if (i != m-1) cout << ' ';
		}

	}
	for(int i=start; i <=n; i++){

	return 0;
}
```

**소스코드 3**    
```c++   
#include <iostream>
using namespace std;
int cnt[10];
void go(int index, int selected, int n, int m) {
    if (selected == m) {
        for (int i=1; i<=n; i++) {
            for (int j=1; j<=cnt[i]; j++) {
                cout << i << ' ';
            }
        }
        cout << '\n';
        return;
    }
    if (index > n) return;
    for (int i=m-selected; i>=1; i--) { // 선택하는데 몇개 선택할거냐 까지 
        cnt[index] = i;
        go(index+1, selected+i, n, m);
    }
    cnt[index] = 0;
    go(index+1, selected, n, m);
}
int main() {
    int n, m;
    cin >> n >> m;
    go(1, 0, n, m);
    return 0;
}
```   
* 각각의 자연수를 선택하는 경우와 선택하지 않는 경우가 있다.      
* 하지만, 중복 선택이 가능하기 때문에, 선택하는 경우르 i개를 선택하는 경우로 세분화 해야한다.

# 5. N 과 M (5)   
* N개의 서로 다른 자연수 중에서 M개를 고른 수열을 모두 구하는 문제       
* 1 <= M <= N <=8   

```c++
#include <iostream>
#include <algorithm>
#include <vector>
#include <string>
#include <stack>
#include <queue>
#include <deque>

using namespace std;

bool check[10];
int answer[10]; // num의 인덱스를 저장하기 위한 배열이다.   
int num[10];

void go(int index, int n, int m){
	if(index == m){
		for(int i=0; i < m ;i++){
			cout << num[answer[i]];
            if (i != m-1) cout << ' ';
		}
		cout << "\n";
		return; // 탈출 조건
	}
	for(int i=0; i<n; i++){ // 반복문에서 index는 같은 값으로 무조건 n만큼 돈다.   
		if(check[i]) continue; // num의 인덱스가 이미 사용되었다면 넘기기
		check[i] =  true; // true 
		answer[index] = i; // num에서 사용할 인덱스를 저장 -> 후에 num[a[]]로 사용   
		go(index+1,n,m);  // index 증가루트 
		check[num[i]] =  false; // 다음을 위한 false 
	}
}

int main(void){
	
	int n, m;
	cin >> n >> m;

	for(int i=0; i<n; i++){
		cin >> num[i];
	}
	sort(num,num+n);
	go(0,n,m);

	return 0;
}
```
